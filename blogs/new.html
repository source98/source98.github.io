    手写new函数

    Zeal 6/22/2024 JavaScript手写

    # new函数是什么？

    new运算符是用来实例化一个类，从而在内存中分配一个实例对象。但在javascript中，万物皆对象，为什么还要通过new来产生对象呢？

    # new函数是怎么运行的？

    其实new函数很简单，如果你了解JavaScript中的原型与原型链就知道new函数无非就是做了以下几个步骤

    1. 创建一个新对象
    2. 将构造函数的作用域赋给新对象
    3. 执行构造函数中的代码
    4. 返回这个新对象

    # 实现手写

    function myNew(Fn, ...args) {
  let object = Object.create(Fn.prototype);
  const result = Fn.apply(object, args);
  return typeof result === 'object' && result !== null ? result : object;
}

    下面我们逐一解读下代码

    • 创建一个对象，继承构造函数的原型。
    let object = Object.create(Fn.prototype);

    我们知道使用构造函数创建实例的时候，实例的隐式原型指向的是构造函数的原型对象。因此上面这段代码我们可以拆解成如下所示。(Object.create方法原理和使用请移步阅读！)

    let obj = {};
obj.__proto__ === Fn.prototype;
    • 调用构造函数，绑定this到新创建的对象，并传入参数。(this指向和绑定请移步阅读！)
    const result = Fn.apply(object, args);
    • 如果构造函数返回一个对象，则返回这个对象；否则返回新创建的对象。 为什么要加入这个条件呢？因为当一个函数返回的是一个对象时，this会指向这个对象。
    return typeof result === 'object' && result !== null ? result : object;

    # 检验代码

    function Person(name) {
  this.name = name;
}

Person.prototype.say = function() {
  console.log(`Hello, my name is ${this.name}`);
};

const jack = myNew(Person, 'jack');
jack.say(); // Hello, my name is jack`
